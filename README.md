# RIQVA - Innovative Solutions

A professional, polished website for RIQVA Software Solutions company, showcasing cutting-edge web development with modern animations and interactive features.

## Features

### 🎨 Design
- **Navy Blue Accent Theme**: Professional color scheme with gradients
- **Fully Responsive**: Optimized for all devices and screen sizes
- **Modern UI/UX**: Clean, intuitive interface with smooth transitions

### ✨ Animations & Effects
- **Loading Animation**: Eye-catching logo reveal on page load
- **Parallax Effects**: Dynamic background animations throughout
- **Scroll Animations**: Elements animate into view as you scroll
- **Hover Effects**: Interactive feedback on all clickable elements
- **Carousel**: Smooth testimonial slider with auto-play

### 📱 Sections
1. **Hero Section**: Compelling introduction with animated background
2. **About Section**: Company overview with key features
3. **Services Section**: 6 comprehensive services with images and details
   - Website Development
   - App Development
   - Business Software (Accounting & Marketing)
   - Digital Marketing
   - Annual Maintenance
   - UI/UX Development
4. **Why Choose Us**: 6 compelling reasons with icons
5. **Process Timeline**: 4-step development process
6. **Testimonials**: Interactive carousel with client reviews
7. **Contact Form**: Professional contact section with CTA
8. **Footer**: Comprehensive footer with links and newsletter

### 🚀 Technical Features
- **React 19** with TypeScript
- **Framer Motion** for smooth animations
- **Tailwind CSS** for styling
- **Intersection Observer** for scroll-based animations
- **Fully optimized** build with Vite

## Services Offered

- ✅ UI/UX Development (Frontend & Backend)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
